# Technical Documentation

## Project Structure
- index.html: Main HTML structure
- style.css: Styling and layout
- script.js: Interactivity (Read More toggle, menu behavior)
- assets/: Images and CV file

## Features
- Responsive layout using Flexbox and media queries
- Smooth scrolling navigation
- Interactive Read More button
- Contact form UI

## Known Limitations
- No backend for form submission
- Static portfolio (front-end only)
